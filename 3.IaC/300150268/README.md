## Introduction

L’Infrastructure as Code (IaC) est une méthode qui permet de gérer et déployer l’infrastructure informatique à l’aide de code plutôt que par des actions manuelles.

Traditionnellement, les systèmes étaient configurés manuellement, ce qui pouvait provoquer des erreurs et des incohérences entre les serveurs.

Avec l’IaC, toute l’infrastructure est décrite dans des fichiers de configuration qui peuvent être versionnés avec Git.

---

## Avantages de l’IaC

- Automatisation des déploiements
- Reproductibilité des environnements
- Réduction des erreurs humaines
- Historique des changements avec Git
- Déploiement rapide et fiable
